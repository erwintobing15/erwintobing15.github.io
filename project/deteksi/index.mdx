## <center>Deteksi Pantun Puisi</center>
<br/>

#### Description
This is a machine learning web based application that can detect type of indonesian poem between pantun and puisi from a text poem. This project was built using django and trained SVM machine learning model

#### Features
* Classifying if an Indonesian poem is a pantun or puisi
* Get probability of classification

#### Technologies
* Python
* Django
* Scikit-learn
* Bootstrap
