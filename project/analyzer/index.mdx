## <center> C++ Analyzer </center>
<br/>

#### Description
A webapp that generate tokens from a C++ source code as object language and implemented using PHP language


#### Features
* Code editor to input object language C++
* Generate tokens from C++ source code
* Classify tokens type as identifier, number, string, operator etc
* Save every token line number in source code
* Count total tokens generate from the source code

#### Technologies
* PHP
* HTML
* JS
* CSS
* Bootstrap
* JQuery
